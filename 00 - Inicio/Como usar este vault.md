# Cómo usar este vault

## Primera vez en el proyecto

1. Empezá por [[Vision del producto]] para entender qué estamos construyendo
2. Leé [[Clean Architecture]] para entender cómo está organizado el código
3. Leé [[Multi-tenant Strategy]] para entender el aislamiento de datos
4. Revisá los [[05 - Decisiones ADR/ADR-001 Multi-tenant por Schema|ADRs]] para entender por qué tomamos cada decisión

## Cómo encontrar lo que buscás

- **Buscar algo específico**: `Ctrl+Shift+F` — búsqueda full-text en todo el vault
- **Navegar entre notas**: `Ctrl+O` — quick switcher
- **Ver conexiones**: Graph view (`Ctrl+G`) — mapa visual de cómo se conectan los conceptos
- **Ver backlinks**: panel derecho → Backlinks

## Cómo contribuir

- Usá los templates en `_templates/` para crear notas nuevas
- Siempre agregá frontmatter con `tags` y `status`
- Linkeá a otras notas con `[[nombre de la nota]]`
- Si tomamos una decisión técnica importante → crear un ADR en `05 - Decisiones ADR/`
- Después de cada reunión → crear una nota en `06 - Reuniones/` usando el template
